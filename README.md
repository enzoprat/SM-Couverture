# SM Couverture

Site vitrine one-page pour **SM Couverture**, artisan couvreur zingueur à Pau (64).

## Stack

- HTML / CSS / JS vanille (un seul fichier `index.html`)
- Google Fonts : Montserrat + Source Sans 3
- Formulaire de contact : [Web3Forms](https://web3forms.com) (envoi AJAX vers `semmedou@icloud.com`)
- Hébergement : Vercel (fichier statique pur, aucun build)

## Structure

```
.
├── index.html
├── img/
│   ├── logo.png
│   ├── photo-1.jpg ... photo-5.jpg
└── README.md
```

## Déploiement Vercel

```bash
vercel --prod
```

Ou bien : connecter le repo GitHub à Vercel via l'interface.
